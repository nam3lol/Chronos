# Chronos

## A simple, yet powerful text editor software.

*Simple, beautiful, lightweight, efficient, reliable.*
These are all words that describe Chronos.

Chronos is a text editor that is designed to be simple but efficient, being packed with only the essential features so you won't be stressed out.

# Features planned
1. Support for *Windows*, *Mac*, and *Linux*
2. Multiple editions *(etc: ***Lite***, ***Classic***, ***Deluxe***)*, with **Lite** having only the essential features and **Deluxe** having advanced features, such as AI writing tools)
3. *to be added*
